# Netflix Clone

Welcome to the **Netflix Clone** project! This project is a modern and responsive user interface inspired by the Netflix homepage. It was built using **HTML5** and **CSS3** to practice frontend layout design and responsive web development.

## 🎯 Project Overview

This project recreates the visual structure of Netflix's landing page and focuses on modern UI design principles.

With this project you can explore:

- **Modern Layout Design:** Netflix-inspired homepage structure
- **Responsive Interface:** Works across desktop, tablet, and mobile screens
- **Clean UI:** Organized sections and visually appealing components
- **CSS Animations:** Smooth hover effects and transitions
- **Semantic HTML:** Well-structured and accessible markup

## 🚀 Features

- Netflix-style hero banner
- Movie/series card layout
- Hover animation effects
- Responsive grid structure
- Clean and modern UI
- Organized semantic HTML5 structure

## 🛠️ Technologies Used

- **HTML5** – Semantic page structure
- **CSS3** – Styling and layout
- **Flexbox** – Responsive layout system
- **CSS Animations & Hover Effects**
